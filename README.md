# Subscription Management Dashboard – Frontend

This project includes the core features of a SaaS-style system such as login, registration, listing plans, a user dashboard, and an admin section.

It is built using React, Parcel, Redux Toolkit, React Router, and plain CSS for styling.

The aim was to create a clean and responsive UI that can easily connect to any backend API.

---

##  Features

###  Authentication
- Login page  
- Register page  
- Logout  
- User token & role managed through Redux

###  Dashboard
- Shows logged-in user’s subscription details (mock data)

###  Plans Page
- Displays list of available subscription plans   

###  Admin Dashboard
- Shows list of all subscriptions (mock data)  
- Only accessible to Admin users

###  Navigation

- Dynamic navbar updates based on login status  
- Admin-only link visible when role is admin


### Styling

- Styled using plain CSS 
- Clean layout with gradient background image  
- Fully responsive for desktop and mobile
      

##  Tech Stack

- **React.js**
- **Parcel Bundler**
- **Redux Toolkit**
- **React Router **
- **Plain CSS**
- **CodeSandbox (for development)**
